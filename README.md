# БЕДОЛАГА

Браузерная казуальная игра для **Яндекс Игр**: кликер + квиз «Кто хочет стать миллионером» + диалоговая «болталка». Главный герой — офисный клерк, который делает карьеру у привлекательной начальницы Вероники: от стажёра до партнёра. Подача — в духе студии Playrix (тёплый глянцевый интерфейс, сочные кнопки, крупный персонаж на фоне офиса).

Игра — статический сайт без сборки: один `index.html` (HTML + CSS + JS) и папка `assets/` с графикой.

## Запуск локально

- Самый простой способ — открыть `index.html` в браузере.
- Лучше через локальный сервер (корректно грузятся картинки из `assets/`):
  ```bash
  python3 -m http.server 8000
  # затем открыть http://localhost:8000
  ```

## Что уже реализовано

- **Кликер-экономика:** 6 генераторов дохода, формулы стоимости/дохода, доход за клик, офлайн-доход (50%, кэп 4 ч).
- **Карьера:** 5 этапов (Стажёр → Исполнительный директор / Партнёр), множитель дохода ×1.0 → ×7.0.
- **Квиз:** 50 бизнес-вопросов, 3 «офисные» подсказки (Отчёт = 50/50, Коллеги, Ментор); верные ответы растят «доверие» → повышение.
- **Болталка:** 10 уровней близости, 30 личных вопросов × 3 варианта ответа (90 реакций), кнопка «Поговорить» + авто-инициатива начальницы.
- **Реплики начальницы:** 120+ вариативных фраз (рост бизнес-показателей + двусмысленные намёки на симпатию), меняются по этапам; фоновые (idle) реплики между событиями.
- **Героиня крупным планом** на фоне офиса: меняет позы, эмоции и реплики (комикс-облачка).
- **Экран меню/старта**, сохранение прогресса (safeStorage → localStorage → память), интеграция Яндекс SDK с фолбэком в автономный режим.
- **UI в стиле Playrix**, горизонтальная раскладка под десктоп + адаптив под узкие экраны.

## Где что настраивать (всё в `index.html`)

- `ASSETS` — слоты графики: фоны офиса, образы начальницы по этапам, слоты `hero` / `desk` под будущие PNG.
- `GENERATORS`, `OFFLINE`, `CLICK_BASE`, `DT_CLAMP` — экономика и баланс.
- `STAGES` — карьерные этапы: множители, мотивации, наборы фраз (`correct` / `wrong`), эмоции.
- `QUESTIONS` — банк бизнес-вопросов квиза (сейчас 50).
- `CHAT`, `CHAT_TITLES`, `CHAT_NEED` — болталка: уровни, вопросы/ответы, темп набора близости.
- `IDLE` — фоновые реплики начальницы.

## Как добавить графику

- **Новые образы начальницы по этапам.** Сейчас один образ (бордовый костюм) на всех этапах: `boss: [BOSS_POSES, BOSS_POSES, …]`. Для уникальных нарядов на этапы 2–5 создайте отдельные наборы (та же раскадровка эмоций) и подставьте в соответствующий индекс `ASSETS.boss[i]`.
- **Образ героя / стол.** Заполните `ASSETS.hero[i]` / `ASSETS.desk[i]` путями к PNG (сейчас там заглушки-эмодзи).
- **Фоны.** Замените файлы `assets/office_*.jpg` или пути в `ASSETS.office`.

## Деплой на Яндекс Игры

1. Заархивировать содержимое проекта (`index.html` + `assets/`) в zip.
2. Загрузить в Консоли разработчика Яндекс Игр.
3. SDK подключается автоматически; в коде уже есть обязательный `LoadingAPI.ready()`, обработка пауз рекламы и облачные сейвы игрока.
4. Для лидерборда — создать его в Консоли с техническим именем `capital` (или поменять имя в функции `submitScore`).

## Примечание про ассеты

В этом репозитории графика — отдельными файлами в `assets/`, а `index.html` ссылается на них (удобно для git и редактирования в Claude Code). Ранее существовал и вариант со всеми картинками, вшитыми в HTML как base64 (один самодостаточный файл, удобный для мгновенного превью) — при необходимости легко собрать обратно из этих же ассетов.
